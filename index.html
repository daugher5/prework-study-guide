Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creats a new branch and switches to it.
  • The .git directory, directly where you initialized your repository, contains all your local Git data.
  • Fetch is a command which gets data about remote branches.
  • Merge brings the remote changes to your local respository.
